## キーワード競合分析（SERP分析）とは？

キーワード競合分析は、指定したキーワードでGoogle検索した際に表示される上位サイトを分析し、競合サイトの強みや特徴を数値で把握できるSEOツールです。

## 🎯 機能概要

![キーワード競合分析 SERP分析結果 Jp](/images/キーワード競合分析-SERP分析結果.jpg)

キーワード競合分析では、以下の3つのステップで包括的な分析を実現します。

### 1. SERP結果の取得（10ポイント）

指定したキーワードでGoogle検索した際に表示される**上位100サイト**を取得します。

**取得できる情報：**

- 検索順位
- タイトル・URL
- 説明文
- 流入予測（ETV：Estimated Traffic Value）
- サイトリンク

### 2. 詳細メトリクスの取得（10ポイント）

各サイトの詳細なSEO指標を取得します。

**取得できる情報：**

- **ページランク**: 個別ページの権威性（0-100）
- **ドメインランク**: サイト全体の権威性（0-100）
- **被リンク数**: そのページへのリンクの総数
- **参照ドメイン**: 被リンクを提供している異なるドメインの数
- **スパムスコア**: 被リンクの品質指標（0-100、低いほど良い）

### 3. AI包括レポートの生成（20ポイント）

AIが分析結果を総合的に評価し、戦略的なレポートを自動生成します。

![キーワード競合分析 レポート Jp](/images/キーワード競合分析-レポート.jpg)

**レポート内容：**

- 競合サイトの強み・弱みの分析
- 自サイトの改善ポイント
- 上位表示に向けた具体的なアクション

## 📖 使い方

### ステップ1: 分析したいキーワードを入力

1. [キーワード競合分析ページ](https://linksurge.jp/simple-serp)にアクセス
2. 分析したいキーワードを入力欄に入力
3. 地域・言語・デバイスを選択（デフォルト設定で問題ありません）
4. 「分析を実行」ボタンをクリック

**ポイント消費**: 10ポイント

### ステップ2: 詳細メトリクスを取得（オプション）

SERP結果が表示されたら、各レコードの**「全件の詳細を取得(10pt)」**ボタンをクリックすると、全サイトの詳細メトリクスを一括取得できます。

**ポイント消費**: 10ポイント

### ステップ3: AIレポートを生成（オプション）

詳細メトリクスを取得した後、**「AI包括レポートを生成(20pt)」**ボタンをクリックすると、AIが分析結果を総合的に評価したレポートを生成します。

**ポイント消費**: 20ポイント

## 📊 分析結果の見方

### キーワード指標カード

分析結果の上部に表示されるカードで、キーワードの基本情報を確認できます。

- **検索ボリューム**: 月間の検索回数
- **CPC**: クリック単価（広告出稿時の参考値）
- **競合度**: LOW（低）・MEDIUM（中）・HIGH（高）
- **難易度**: 0-100のスコア（高いほど上位表示が困難）

### SERP分析結果テーブル

検索結果上位10サイトの詳細情報がテーブル形式で表示されます。

#### 各項目の説明

| **順位**       | Google検索結果での表示順位（1位が最も上位）   |
| ------------ | --------------------------- |
| **タイトル・URL** | 検索結果に表示されるタイトルとURL          |
| **流入予測**     | その順位で獲得できると予測される月間アクセス数     |
| **ページランク**   | 個別ページの権威性（詳細取得後）            |
| **ドメインランク**  | サイト全体の権威性（詳細取得後）            |
| **被リンク数**    | そのページへのリンクの総数（詳細取得後）        |
| **参照ドメイン**   | 被リンクを提供している異なるドメインの数（詳細取得後） |
| **スパムスコア**   | 被リンクの品質指標（詳細取得後、80以上は要注意）   |

#### ツールチップ（情報アイコン）

各項目名の横にある「ℹ️」アイコンにマウスをホバーすると、その項目の詳細な説明が表示されます。

### CSV出力機能

分析結果をCSVファイルとしてダウンロードできます。

- **SERP結果CSV**: 基本情報（順位・タイトル・URL・流入予測など）
- **詳細結果CSV**: 詳細メトリクスを含む完全なデータ

## 💰 ポイント消費について

キーワード競合分析では、以下のポイントが消費されます。

| 機能          | ポイント消費 |
| ----------- | ------ |
| SERP結果の取得   | 10ポイント |
| 詳細メトリクスの取得  | 10ポイント |
| AI包括レポートの生成 | 20ポイント |

**合計**: 最大40ポイント（全ての機能を使用した場合）

**ポイント節約のコツ：**

- 詳細メトリクスやAIレポートは必要に応じて取得する
- まずはSERP結果だけを確認して、必要に応じて詳細を取得する

## 📝 分析結果の保存・共有

### CSV出力の活用

分析結果をCSV出力することで、以下のような活用が可能です。

- **Excel/Googleスプレッドシートでの分析**: データの並び替え・フィルタリング・グラフ作成

- **社内共有**: チームメンバーと分析結果を共有

- **レポート作成**: 分析結果をレポート形式でまとめる

- **経時比較**: 定期的に分析して、順位の変化を追跡

### 分析履歴の活用

**「最近の分析結果を見る」**から過去12時間以内の分析結果を確認できます。

- **経時比較**: 同じキーワードを定期的に分析して、順位の変化を追跡

- **過去の分析結果の再確認**: 過去の分析結果を再確認して、新しい気づきを得る

## ❓ よくある質問

### Q1. どのキーワードを分析すべきですか？

**A.** 自サイトで上位表示を目指しているキーワード、または競合サイトが上位表示しているキーワードを分析することをおすすめします。

### Q2. 詳細メトリクスは必ず取得する必要がありますか？

**A.** いいえ、必須ではありません。SERP結果の基本情報だけでも、競合サイトの状況は十分に把握できます。詳細メトリクスは、より深い分析が必要な場合に取得してください。

### Q3. AIレポートはどのように活用すればいいですか？

**A.** AIレポートは、分析結果を総合的に評価し、具体的な改善アクションを提案します。SEO戦略の立案や、競合サイトとの差を埋めるための参考としてご活用ください。

### Q4. 分析結果は保存されますか？

**A.** はい、分析結果は12時間保存されます。「最近の分析結果を見る」から過去の分析結果を確認できます。

### Q5. CSV出力は何に使えますか？

**A.** CSVファイルは、ExcelやGoogleスプレッドシートで開いて、データの並び替えやフィルタリング、グラフ作成などに活用できます。レポート作成や社内共有にも便利です。

### Q6. スパムスコアが高いサイトは要注意ですか？

**A.** はい、スパムスコアが80以上のサイトは、低品質な被リンクが多い可能性があります。ただし、スパムスコアだけで判断せず、他の指標（ドメインランク・被リンク数など）も総合的に見て判断してください。

### Q7. 流入予測（ETV）はどのように計算されますか？

**A.** 検索順位と検索ボリュームから、その順位で獲得できると予測される月間アクセス数を算出しています。実際のアクセス数とは異なる場合があります。

### Q8. 分析に時間がかかりますか？

**A.** SERP結果の取得は通常10-30秒程度です。詳細メトリクスの取得は、サイト数によって異なりますが、1-2分程度かかる場合があります。
